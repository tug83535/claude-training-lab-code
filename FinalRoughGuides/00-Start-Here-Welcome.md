## What Is the P&L Automation Toolkit?

A single Excel file that automates the entire monthly P&L close process — data import, reconciliation, variance analysis, dashboards, PDF reporting, version control, and more — through a 62-action Command Center. No software to install. No subscriptions. Just open the file and press Ctrl + Shift + M.

## What Is the Universal Toolkit?

A separate collection of 100+ tools (79 VBA macros + 22 Python scripts) that work on **any** Excel file — not just the P&L file. Data cleaning, auditing, formatting, finance tools, duplicate detection, file comparison, fuzzy matching, PDF extraction, and more. See Guide 06 for the full list.

---

## Quick Start (3 Steps)

1. **Save** `ExcelDemoFile_adv.xlsm` to a folder on your computer
2. **Open** the file in Excel and click **Enable Content** on the yellow security bar
3. **Press Ctrl + Shift + M** — the Command Center opens. Pick an action and click Run.

That's it. For the full setup walkthrough (Trust Center settings, troubleshooting), see **Guide 02**.
