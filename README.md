# APCLDmerge — P&L Demo Project
### iPipeline Finance & Accounting | Built for 2,000+ Employees

---

## About This Project

This repository contains all files for a world-class Profit & Loss (P&L) demo
built for iPipeline's Finance & Accounting department. The project combines an
Excel workbook with VBA macros, SQL scripts, and Python automation to deliver a
polished, fully interactive financial demo designed to be presented to the
entire company, including the CFO and CEO.

A video walkthrough is also planned to help coworkers understand and use the
tools included in this project.

---

## What's Inside

| Folder | Purpose |
|--------|---------|
| `vba/` | VBA modules (.bas files) — the macros that power the Excel workbook |
| `sql/` | SQL scripts for data staging, transformation, validation, and enhancements |
| `python/` | Python scripts for automation, reporting, forecasting, and testing |
| `excel/` | The main demo P&L Excel file — uploaded and reviewed each session |
| `docs/day-to-day/` | Quick-reference guides for daily operations |
| `docs/overview/` | High-level project architecture and executive summary |
| `docs/setup/` | Step-by-step setup and implementation guides |
| `training/` | Final, reviewed training materials ready for coworkers |
| `qa/` | QA tracking, test plans, checklists, and bug/issue logs |
| `tasks/` | Session management — todo.md (task list) and lessons.md (lessons learned) |
| `DemofileChartBuild/` | Chart sheet files for the demo P&L workbook, work in progress |
| `NewTesting/` | Experimental code, research, and ideas not yet ready for the main project |
| `CompletePackageStorage/` | Final, production-ready files and versioned backups |
| `CompletePackageStorage/production/` | Live, ready-to-go final files |
| `CompletePackageStorage/backups/` | Versioned backups of completed work |
