# iPipeline Video 4 Handoff Pack

This folder contains four handoff files created from the external review of the iPipeline Finance Automation Demo Project overview.

## Files

1. `01_claude_code_review_brief.md`  
   A review brief for Claude Code. Use this to ask Claude Code to reassess the project plan, reconcile stale assumptions, and produce updated planning docs.

2. `02_codex_build_spec.md`  
   A build specification for Codex. Use this to build the safe local Python Video 4 package: Revenue Leakage Finder, Data Contract Checker, Exception Triage Engine, Control Evidence Pack, Workbook Dependency Scanner, and Finance Automation Launcher.

3. `03_explain_like_im_5.md`  
   A simple plain-English explanation of the plan.

4. `04_release_safety_distribution_checklist.md`  
   Extra recommended file. This covers the missing release/safety/distribution layer: SharePoint package, safety gates, support model, pilot plan, and release checklist.

## Core recommendation

Lock one official Video 4. Make Revenue Leakage Finder the hero. Use Data Contract Checker, Exception Triage, and Control Evidence Pack as supporting demos. Ship a safe local Python Automation Launcher. Narrow the public entry point to 5–7 supported workflows. Use controlled SharePoint distribution for 50–150 coworkers. Park xlwings, external AI, and enterprise deployment for v2.
