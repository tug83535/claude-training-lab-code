PDF EXTRACTOR DEMO
==================

You need a PDF file with data tables for the demo.

Best options:
1. Any financial statement PDF (quarterly report, 10-K, etc.)
2. An invoice summary PDF
3. A vendor report PDF

Requirements:
- Must have SELECTABLE TEXT (not a scanned image)
- Must have at least one visible table with rows and columns
- Save it to this folder as 'sample_report.pdf'

The script command will be:
  python pdf_extractor.py sample_report.pdf
