## Full Narration Script

### 00:00–00:20 Hook
"What if month-end prep could go from manual tab-hopping to a single guided run? In this demo, one click will run controls, write commentary, and generate an executive brief pack."

### 00:20–01:20 Problem framing
"Finance teams spend too much time stitching together checks, commentary, and output packs. The challenge is not just calculations. The challenge is repeatability, auditability, and clear outputs for leadership. Today we show a workflow that handles all three."

### 01:20–03:00 Live one-click workflow highlight
"We start on the Demo Command Center. First, run reconciliation. The checks sheet updates with PASS and FAIL status. Next, generate variance narrative. A new sheet appears with plain-English movement explanations. Then run scenario comparison for base and stress outcomes. Finally, build the executive brief pack and export a PDF."

### 03:00–05:20 Result walkthrough
"Now we open each artifact. The Checks sheet confirms source completeness and tie-outs. The variance narrative sheet converts raw movement into executive-ready language. The scenario sheet gives decision-ready range analysis. The executive brief consolidates KPI, controls, and trend movement in one branded output."

### 05:20–06:30 Business impact
"This reduces repetitive month-end effort and improves consistency. The team gets faster reviews, fewer manual misses, and a clear audit path. Executives receive the same trusted output format each cycle."
