## Full Narration Script

### 00:00–00:20 Hook
"No custom model required. We will drop the universal toolkit into a standard workbook and run high-value automation in minutes."

### 00:20–01:30 Setup
"We open a plain quarterly report workbook and run `BuildCommandCenter`. This creates a reusable launch surface with one-click actions."

### 01:30–03:10 Data sanitizer + profile
"First, preview sanitizer impact, then run full sanitize. This normalizes mixed dates, text-stored numbers, and spacing issues. Next, run workbook profile to map header rows and range boundaries so downstream steps run consistently."

### 03:10–04:50 Compare and consolidate
"Now run sheet compare to identify changed rows and build a difference report. Then consolidate visible sheets by header so analysis can happen on one consistent table with source trace columns."

### 04:50–06:10 Intelligence tools
"Run materiality classifier and exception narratives. The toolkit tags high-risk movement and writes plain-English comments that a reviewer can use immediately."

### 06:10–07:20 Output pack
"Finally, build the executive one-pager. This creates a quick KPI summary that can be shared as-is or exported as a PDF in one step."

### 07:20–08:00 CTA
"If your workbook is different, use the CoPilot Prompt Guide to adapt file-specific demos safely. Start with one module, validate outputs, then scale."
