## Full Narration Script

### 00:00–00:20 Hook
"Your workbook does not need to match the demo workbook. You can adapt the same logic safely with a structured CoPilot workflow."

### 00:20–01:20 Workbook mapping method
"Start by mapping your sheets, header rows, key columns, named ranges, and business rules in plain language. This is the most important step because it anchors CoPilot to your actual structure."

### 01:20–03:10 Starter prompt live run
"Now paste the starter prompt from the CoPilot guide, followed by your workbook map and one module. Ask CoPilot to return changed procedures only. Apply changes in a test copy, not production."

### 03:10–04:30 Validation checklist
"Run the adapted macro once. Check output sheet names, PASS/FAIL messaging, and audit log updates. Spot-check key values manually before accepting the change."

### 04:30–05:30 Troubleshooting loop
"If CoPilot changed too much, ask for sheet and column remap only. If logic drifted, request a rollback to original behavior with minimal mapping edits. Iterate in small steps until results are stable."

### 05:30–06:00 Final CTA
"Use this method to adapt one feature this week, validate it, and then scale to the full toolkit."

## On-Screen Action Callouts

## Closing CTA
"Start with one adaptation, document it in your change log, and expand only after a clean validation pass."
