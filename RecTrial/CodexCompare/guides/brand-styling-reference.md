# Brand Styling Reference (Operational)

## 1) Color palette

- iPipeline Blue: `#0B4779`
- Navy: `#112E51`
- Innovation Blue: `#4B9BCB`
- Lime Green: `#BFF18C`
- Aqua: `#2BCCD3`
- Arctic White: `#F9F9F9`
- Charcoal: `#161616`

## 2) Fonts

- Use **Arial only**.
- Headings: 18–24 pt
- Subheadings: 14–16 pt
- Body: 10–11 pt

## 3) Layout standards

- Add branded header block on every report sheet.
- Use thin light-gray borders.
- Keep number formats consistent.

## 4) Chart standards

- No 3D charts.
- First series = iPipeline Blue.
- Second series = Innovation Blue.
- Third series = Aqua.

## 5) PDF standards

- Header on each page.
- Footer text: `Confidential — iPipeline Finance & Accounting`.
- 0.5" margins.

## 6) Banned items

- Default rainbow palettes
- WordArt
- Comic Sans
- Emoji in official outputs
