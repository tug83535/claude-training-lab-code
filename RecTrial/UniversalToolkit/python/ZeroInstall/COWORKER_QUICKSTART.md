# COWORKER QUICKSTART (Finance & Accounting)

Version: 1.0.0

## Step 1
Click the **Excel button** labeled **Finance Automation Launcher** in the demo workbook.

## Step 2
When the launcher window/menu appears, choose a menu number from 1 to 6.

## Step 3
Read the short script description shown in the menu.

## Step 4
Type the same script manually in Command Prompt with `--help` to see exactly what file it expects.

## Step 5
Locate your input file in File Explorer and copy its full path.

## Step 6
Run the script command with your input path and an output path.

## Step 7
If you see an error saying a file path is missing or not found, stop and fix the path, then rerun.

## Step 8
Open the output file and confirm it looks correct before sharing.

## Step 9
Keep your original source file unchanged. Work from copies.

## Step 10
If unsure, send the script name, command, and error text to the project owner.
