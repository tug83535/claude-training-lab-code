# Demo Story Recommendation (Coworker + Leadership)

## Purpose
This recommendation is tailored to the project goal in `CLAUDE.md`: show coworkers and leadership what is possible when VBA, SQL, and Python work together in one practical Finance workflow.

## Core Recommendation
Lead with a **single business story** instead of a feature tour:

1. **Start in VBA (Excel Command Center):** trigger the close workflow from a familiar interface.
2. **Show SQL in the middle:** explain that raw rows are normalized/structured so totals stay consistent and traceable.
3. **Finish with Python outputs:** forecasting, validation summaries, and presentation-ready artifacts.

This sequence keeps non-technical viewers oriented: **"click in Excel -> data confidence from SQL -> advanced insights from Python."**

## 10-Minute Live Demo Script

### Minute 0-1: Business framing
- State the problem in plain language: month-end is slow, manual, and hard to audit.
- State the promise: one repeatable flow that is faster and cleaner.

### Minute 1-4: VBA value
- Open Command Center.
- Run 2-3 high-impact commands (import/reconcile/dashboard).
- Emphasize that finance users do not need to code to use this.

### Minute 4-6: SQL value
- Show one staging/fact-view concept visually (before vs after structure).
- Emphasize trust: standardized joins/dedup logic reduce reporting mistakes.

### Minute 6-8: Python value
- Run one forecast/simulation/report action.
- Show output artifact (table/chart/report) and explain why this is difficult in VBA alone.

### Minute 8-10: Outcome + call to action
- Summarize benefits: speed, consistency, audit trail, leadership-ready output.
- Ask coworkers to pilot with one monthly cycle and provide feedback.

## What to Avoid
- Avoid listing dozens of commands without context.
- Avoid deep technical internals unless asked.
- Avoid switching files too often; keep one clear narrative.

## "So What" Slide (Recommended)
End with a single slide:
- **VBA:** user-friendly workflow entry point in Excel.
- **SQL:** data integrity and repeatable structure.
- **Python:** advanced analytics and automation.
- **Result:** less manual effort, better confidence, faster decisions.

## Next Step Recommendation
Build one polished "hero" scenario for the coworker walkthrough video:
- Start with messy monthly data.
- Run the integrated workflow.
- End with an executive-ready result and audit trace.

If this hero scenario lands, interest in the broader toolset will follow naturally.
