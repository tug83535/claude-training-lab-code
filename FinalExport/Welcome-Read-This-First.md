## 3-Step Quick Start

1. Open the **START HERE** folder and read the setup guide
2. Download the Excel file from the **Demo File** folder
3. Open it, enable macros, and press **Ctrl + Shift + M**

That's it. The Command Center opens and you're ready to go.
